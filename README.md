# SoulFireClient

This is a frontend for the SoulFire server.
It mainly targets the web, but uses native APIs using Tauri.

Tools used:

- Vite
- tanstack-router
- Tauri
- shadcn/ui
- TailwindCSS
- XTerm
- protobuf-ts
- and more...

## Linux dependencies

```bash
sudo apt install libavahi-client-dev
```
